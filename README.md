## Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio profissional, desenvolvido em React com TypeScript. O objetivo é apresentar meus principais projetos, habilidades e experiências como desenvolvedor front end.

## Tecnologias Utilizadas
- React
- TypeScript
- Styled-components
- Vite ou Create React App
- ESLint & Prettier
- Testes com Jest/React Testing Library

## Instalação

1. Clone o repositório:
	```bash
	git clone https://github.com/seu-usuario/portfolio.git
	```
2. Acesse a pasta do projeto:
	```bash
	cd portfolio
	```
3. Instale as dependências do projeto:
	```bash
	npm install
	```
	_ou, se preferir usar yarn:_
	```bash
	yarn install
	```
4. Inicie o projeto em modo desenvolvimento:
	```bash
	npm start
	```
	_ou:_
	```bash
	yarn start
	```

## Como Usar
- Acesse [http://localhost:3000](http://localhost:3000) após rodar o projeto.
- Navegue pelo menu lateral para conhecer meus projetos e informações.
